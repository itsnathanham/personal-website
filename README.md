# personal-website

Single-file React site (no build step) based on the provided design.

## Run locally

From the project root:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize

Edit `index.html` (content, links, styles, pages).

Hero image is `assets/profile.png`.

## Deploy

Any static host works (GitHub Pages, Netlify, Vercel static, Cloudflare Pages).
